# Design token ratios

Design token setup for a typographic scale using standard ratios:

* Minor second (`1.067`)
* Major second (`1.125`)
* Minor third (`1.2`)
* Major third (`1.25`)
* Perfect fourth (`1.333`)
* Augmented fourth (`1.414`)
* Perfect fifth (`1.5`)
* Golden ratio (`1.618`)

Custom ratios can be edited or added.
